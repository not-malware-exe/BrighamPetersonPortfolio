Description

My Asteroids Clone project is a web hosted, 2d, arcade style game.

The game is a asteroids clone that is hosted on the web. You control a ship that only has 2 thrusters and a gun. You have to use your thrusters wisely to keep the ship from drifting into an asteroid or spinning out of control. Use your gun to shoot asteroids to earn points. Earn as many points as possible before your ship crashes. To make this game, I essentially had to make my own game engine in js specifically tailor made for this game.
